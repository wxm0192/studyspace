MySQL 初体验

说明：

本实验搭建一个由 Docker 提供的 MySQL 数据库 。 体验通过 Navicat 客户端连接MySQL 数据库 ， 体验通过SQL 脚本建立数据库 ， 建立表 ， 导入数据 ， 体验 Navicat 查看数据库 。

 

步骤01：网络测试

 

本实验需要访问提供公网的Web 服务，需要测试能够连通公网 。

在此通过测试 8.8.8.8 公网地址验证能否连通公网 。

在命令行输入 ：

#ping 8.8.8.8

如果看到如下显示， 说明能够连通公网 ：

PING 8.8.8.8 (8.8.8.8) 56(84) bytes of data.

64 bytes from 8.8.8.8: icmp_seq=3 ttl=110 time=42.7 ms

 

步骤02：启动docker 服务

 

本实验通过docker 提供 MySQL 服务 ， 需要首先启动 docker 管理进程 。

 

 2、启动docker服务

# systemctl start docker

 

3、验证docker是否启动成功

#ps aux | grep docker

 

显示类似信息：

root   3835  0.2  5.6 578936 56768 ?     Ssl  05:51   0:00 /usr/bin/dockerd -H fd:// --containerd=/run/containerd/containerd.sock

上显示docker环境启动完成。

 

步骤03：检查MySQL 镜像文件

 

1、列出MySQL 的镜像信息

#  docker images

 

REPOSITORY        TAG       IMAGE ID        CREATED        SIZE

mysql              5.7       1d7aba917169   5 months ago    448MB

 

以上可以看出本实验环境里面已经有 MySQL的镜像 。自己建立实验环境时， 可以通过

        docker pull mysql

下载MySQL Docker 镜像 （注意后面的 tag ） 。

 

步骤04：启动 MySQL 容器

 

1、后台启动MySQL 容器

首先建立MySQL 的工作目录， 建立MySQL 配置文件 ：

#  mkdir -p /root/mysql/data  \

/root/mysql/logs               \

/root/mysql/conf                \

/root/mysql/run

# cd /root/mysql/

# chown polkitd:input  *

# chmod 777 *

然后启动MySQL 容器 ：

# docker run -p 3306:3306 --name mysql \

-v /root/mysql/conf:/etc/mysql/conf.d         \

-v /root/mysql/logs:/logs                                  \

-v /root/mysql/data:/var/lib/mysql               \

-v /root/mysql/run:/run/mysqld                      \

-e MYSQL_ROOT_PASSWORD=root            \

--privileged=true -d mysql:5.7

 

其中 ：

-d: 后台运行容器

-p 将容器的端口映射到本机的端口

-v 将主机目录挂载到容器的目录

-e 设置参数

 

步骤05：查看容器运行状态

 

1、查看容器运行状态，status为running为正常。

 

# docker ps --format " \n ID:{{.ID}}\n Image:{{.Image}}\n Command: {{.Command}}\n Status:{{.Status}}"  

 

 可以看到下列类似信息 ：

 ID:506cb88504cd

 Image:mysql:5.7

 Command: "docker-entrypoint.s…"

 Status:Up 35 seconds

 

步骤06：下载数据库客户端工具 Navicat

 

1、  从阿里云 OSS 下载Navicat 数据库客户端工具  

https://wxm-web-ssh-01.oss-cn-beijing.aliyuncs.com/tools/navicat160_premium_cs_x64.exe  （Win 10 平台 ）

或者通过 http://www.navicat.com.cn/download/navicat-premium#win 页面下载不同平台的Navicat 工具

 

步骤07：客户端进行MySQL访问测试

打开安装好的Navicat客户端工具， 输入公网地址 ， 其中 公网地址为实验指导说明上方显示的公网地址 ， 输入用户名 root ， 密码 root ， 进行数据库连接 。

如果看到 数据库信息， 说明Navicat 连接数据库成功 。

 在Webshell 界面 ：

#mkdir  /var/lib/mysql

#ln -s  /root/mysql/run/mysqld.sock  /var/lib/mysql/mysql.sock   

#cd /root/mysql

#wget https://wxm-web-ssh-01.oss-cn-beijing.aliyuncs.com/src/demo.sql

在# 下手动输入 ：

       mysql -hlocalhost -uroot -proot

 

在如下提示符下：

MySQL [(none)]>

输入 source demo.sql

该脚本创建数据库  ， 创建表 ， 并在表中导入两条记录 ， 可以在Navicat 界面中看到这两条记录 。

如果看到 数据库中表的内容， 说明实验成功 。
